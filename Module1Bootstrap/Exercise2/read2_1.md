# Exercise 2.1 - Bootstrap Structure and Files

## Objective

Explore the structure of the downloaded Bootstrap directory and understand the purpose of the CSS, JS, and Icons folders.

## Bootstrap Directory Structure

```text
bootstrap/
│
├── css/
│   ├── bootstrap.css
│   ├── bootstrap.min.css
│   ├── bootstrap.rtl.css
│   └── bootstrap.rtl.min.css
│
├── js/
│   ├── bootstrap.js
│   ├── bootstrap.min.js
│   ├── bootstrap.bundle.js
│   └── bootstrap.bundle.min.js
│
└── icons/
    ├── alarm.svg
    ├── house.svg
    ├── person.svg
    └── ...
```

## CSS Folder

The `css` folder contains Bootstrap stylesheets used for designing and styling web pages.

### Important Files

* **bootstrap.css** – Standard readable CSS file.
* **bootstrap.min.css** – Minified version used in production for faster loading.
* **bootstrap.rtl.css** – Supports right-to-left languages such as Arabic and Hebrew.
* **bootstrap.rtl.min.css** – Minified RTL version.

### Purpose

Provides ready-to-use styling classes for layouts, forms, buttons, navigation bars, cards, grids, and other UI components.

## JS Folder

The `js` folder contains JavaScript files required for Bootstrap's interactive components.

### Important Files

* **bootstrap.js** – Standard JavaScript file.
* **bootstrap.min.js** – Minified version.
* **bootstrap.bundle.js** – Includes Bootstrap JavaScript and Popper.js.
* **bootstrap.bundle.min.js** – Minified bundle version.

### Purpose

Enables interactive features such as:

* Modal windows
* Dropdown menus
* Accordions
* Tooltips
* Popovers
* Offcanvas menus
* Carousels

## Icons Folder

The `icons` folder contains Bootstrap Icons in SVG format.

### Examples

* house.svg
* person.svg
* search.svg
* alarm.svg

### Purpose

Provides scalable vector icons that can be used in buttons, navigation bars, forms, and other UI elements.

## Conclusion

The Bootstrap directory is organized into three main folders:

* **CSS** for styling and layout.
* **JS** for interactive functionality.
* **Icons** for graphical elements.

This structure helps developers quickly build responsive and modern web applications using Bootstrap.
